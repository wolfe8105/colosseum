* What would be required: Building a Roblox-style platform would require: a game engine/editor accessible to users, server infrastructure for hosting user-created worlds, a social/multiplayer networking layer, a virtual economy system, moderation tools, and account management. This is a multi-million dollar development project, not a reskin.
* Recommendation: Skip this entirely for reskinning purposes. If you want a sandbox-style building game, search the Unity Asset Store (assetstore.unity.com) for "voxel world builder" or "sandbox building game" — but expect a development project, not a reskin. Budget $200-$500+ for a quality kit, plus developer time.
