## Agent 01

### showCiteDropdown
This function is synchronous and receives a `CurrentDebate` parameter named `debate`. It begins by calling `hideDropdown()` to clear any currently open dropdown, then looks up the DOM element with id `feed-ref-dropdown`; if that element is absent the function returns immediately. It reads the module-level `loadedRefs` array (imported from `./arena-state.ts`) and filters it to entries whose `cited` flag is falsy, storing the result in `uncited`. If `uncited` is empty it calls `showToast('No references remaining', 'error')` and returns. Otherwise it builds an HTML string that starts with a title div, iterates over each `ref` in `uncited` appending a `.feed-dropdown-item` div carrying a `data-ref-id` attribute set to `escapeHTML(ref.reference_id)`, a `.feed-dropdown-claim` span containing `escapeHTML(ref.claim)` wrapped in quotes, and a `.feed-dropdown-meta` span built from `escapeHTML(ref.author)` and `Number(ref.current_power)` (the numeric cast is applied before interpolation). A trailing `.feed-dropdown-cancel` div with id `feed-dropdown-close` is appended. The function writes this HTML to `dropdown.innerHTML` and sets `dropdown.style.display = 'block'`.

It then attaches event listeners: a click listener on `#feed-dropdown-close` bound directly to `hideDropdown`, and for every `.feed-dropdown-item` a click listener that is itself an `async` function. Inside that per-item listener it reads `(item as HTMLElement).dataset.refId`; if the id is missing it returns. It calls `hideDropdown()` and then enters a `try` block that awaits `citeDebateReference(debate.id, refId, round, debate.role || 'a')` — `round` is imported from `./arena-feed-state.ts` and `debate.role` falls back to the string `'a'` when falsy. On success it maps over `loadedRefs` producing a new array in which the entry whose `reference_id` matches `refId` is replaced by a spread copy with `cited` set to true and `cited_at` set to the current ISO timestamp, passes that array to `set_loadedRefs(updated)`, and calls `updateCiteButtonState()`. The `catch (e)` branch reads `e.message` when `e instanceof Error` else uses the string `'Cite failed'`, and calls `showToast(msg, 'error')`. The outer `showCiteDropdown` itself returns `void` and does not await the listener.

### showChallengeDropdown
This synchronous function accepts `debate: CurrentDebate`. It calls `hideDropdown()`, looks up `feed-ref-dropdown` in the DOM, and returns early if the element is missing. It reads the module-level `opponentCitedRefs` array and filters for entries whose `already_challenged` flag is falsy, storing the result in `challengeable`. Two guard branches follow: if `challengeable.length === 0` it calls `showToast('No references to challenge', 'error')` and returns; if the module-level `challengesRemaining` is less than or equal to zero it calls `showToast('No challenges remaining', 'error')` and returns. Otherwise it builds an HTML string with a title div, then iterates over `challengeable` producing `.feed-dropdown-item.feed-dropdown-challenge` entries with `data-ref-id="${escapeHTML(ref.reference_id)}"`, a quoted `escapeHTML(ref.claim)` span, and a meta span containing `escapeHTML(ref.domain)`. A `.feed-dropdown-cancel` div with id `feed-dropdown-close` is appended. The composed HTML is written to `dropdown.innerHTML` and `dropdown.style.display` is set to `'block'`.

It attaches a click listener on `#feed-dropdown-close` bound to `hideDropdown`, and for each `.feed-dropdown-item` an `async` click listener. In that listener it reads `dataset.refId`; if missing it returns. It calls `hideDropdown()` then `try`-awaits `fileReferenceChallenge(debate.id, refId, round, debate.role || 'a')` and assigns the resolved value to `result`. If `result.blocked` is truthy it calls `showToast('\uD83D\uDEE1\uFE0F Shield blocked the challenge!', 'info')` and does nothing else. Otherwise it calls `set_challengesRemaining` with `result.challenges_remaining ?? (challengesRemaining - 1)`, calls `updateChallengeButtonState()`, calls `set_activeChallengeRefId(refId)`, calls `set_activeChallengeId(result.challenge_id || null)` (commented `F-55: store for rule_on_reference`), and then calls `pauseFeed(debate)` — `pauseFeed` is imported from `./arena-feed-machine-pause.ts` and is invoked without `await`. The `catch (e)` branch extracts `e.message` when `e instanceof Error` else uses `'Challenge failed'`, and calls `showToast(msg, 'error')`. `showChallengeDropdown` itself returns `void`.

### showReferencePopup
This synchronous function takes `el: HTMLElement`. It first calls `document.getElementById('feed-ref-popup')?.remove()` to remove any pre-existing popup element. It then reads four values: `el.dataset.url` (falling back to `''`), `el.dataset.sourceTitle` (falling back to `''`), `el.dataset.sourceType` (falling back to `''`), and `el.textContent?.trim()` (falling back to `''`), assigning them to `url`, `sourceTitle`, `sourceType`, and `claim` respectively. It calls `document.createElement('div')`, assigns `popup.className = 'feed-ref-popup'` and `popup.id = 'feed-ref-popup'`, and writes an HTML template string to `popup.innerHTML` containing: a `.feed-ref-popup-claim` div with `escapeHTML(claim)` in quotes; a `.feed-ref-popup-meta` block with a `.feed-ref-popup-type` span holding `escapeHTML(sourceType.replaceAll('_', ' '))` and a `.feed-ref-popup-domain` span holding `escapeHTML(sourceTitle)`; a conditional anchor rendered only when `url` is truthy, with `href="${escapeHTML(url)}"`, `target="_blank"`, `rel="noopener noreferrer"`, and the text `Open source ↗`; and a `.feed-ref-popup-close` button with id `feed-ref-popup-close`. It appends `popup` to `document.body`.

It then attaches two listeners. The first is a click listener on `#feed-ref-popup-close` whose handler calls `popup.remove()`. The second is a click listener on `popup` itself whose handler checks `e.target === popup` and, if true, calls `popup.remove()` — this makes clicks on the outer popup (but not its inner children) dismiss it. The function returns `void`.

### hideDropdown
The function takes no parameters and is synchronous. It calls `document.getElementById('feed-ref-dropdown')` and stores the result in `dropdown`. If `dropdown` is truthy, it sets `dropdown.style.display = 'none'` and assigns `dropdown.innerHTML = ''` (which clears the element and removes any listeners attached to descendants created by `showCiteDropdown` or `showChallengeDropdown`). If `dropdown` is null the function does nothing. It returns `void` and does not call any other functions on the anchor list or any imported helpers.

### showReferencePopup
The function is synchronous and takes an `HTMLElement` parameter named `el`. It first calls `document.getElementById('feed-ref-popup')?.remove()` to delete any pre-existing popup element with that id. It then reads four values: `el.dataset.url` (defaulting to `''`), `el.dataset.sourceTitle` (defaulting to `''`), `el.dataset.sourceType` (defaulting to `''`), and `el.textContent?.trim()` (defaulting to `''` when textContent is null/undefined), storing them as `url`, `sourceTitle`, `sourceType`, and `claim`. It creates a new `div` via `document.createElement`, assigns `className = 'feed-ref-popup'` and `id = 'feed-ref-popup'`, and sets its `innerHTML` to a template string containing a `feed-ref-popup-inner` wrapper with the escaped `claim`, a meta row with `escapeHTML(sourceType.replaceAll('_', ' '))` and `escapeHTML(sourceTitle)`, a conditional anchor — included only when `url` is truthy — whose `href` is `escapeHTML(url)` and which has `target="_blank"` and `rel="noopener noreferrer"`, and a close button with id `feed-ref-popup-close`.

The function appends the new `popup` to `document.body`, then attaches a click listener to the element with id `feed-ref-popup-close` (via `document.getElementById(...)` and optional chaining) whose callback invokes `popup.remove()`. It also attaches a click listener directly on `popup` itself whose callback checks `e.target === popup` and, when true, calls `popup.remove()` — so clicking the outer container (but not inner children) removes the popup. The function returns `void` and does not call any other function on the anchor list.
