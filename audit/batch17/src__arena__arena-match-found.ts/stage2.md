# Stage 2 Outputs — src/arena/arena-match-found.ts

## Agent 01

### clearQueueTimersInline
Synchronous, no parameters, no return value. Reads module state `queuePollTimer` and `queueElapsedTimer` from `./arena-state.ts`. If `queuePollTimer` is truthy, calls `clearInterval(queuePollTimer)` then writes `null` back via `set_queuePollTimer(null)`. If `queueElapsedTimer` is truthy, calls `clearInterval(queueElapsedTimer)` then writes `null` back via `set_queueElapsedTimer(null)`. No branches beyond those two guards; no error paths.

### onMatchFound
Synchronous entry, but schedules async work via `setTimeout`. Reads parameter `data: MatchData` and module state `selectedMode`, `selectedRanked`, `selectedRuleset` (all from `arena-state.ts`) and the `DEBATE` config plus `AI_TOPICS` constant. Calls `clearQueueTimersInline()` first to stop queue timers. Then reads the DOM element `#arena-queue-status`; if present, writes `'🎯 OPPONENT FOUND!'` to `textContent` and sets inline color to `var(--mod-accent)`. Schedules a 1200 ms `setTimeout` callback that builds a `CurrentDebate` object populated from `data` fields, with each field falling back to a default when missing: `topic` falls back to `randomFrom(AI_TOPICS)`, `role` to `'a'`, `mode` to `selectedMode ?? 'text'`, `totalRounds` to `DEBATE.defaultRounds`, `opponentName` to `'Opponent'`, `opponentId` to `null`, `opponentElo` to `1200`, `ruleset` to `selectedRuleset`, `language` to `'en'`, `messages` to `[]`; `ranked` is copied from `selectedRanked`. Branch inside the timeout: if `selectedMode === 'ai'` or `data.opponent_id` is falsy, calls `enterRoom(debateData)` directly; otherwise calls `showMatchFound(debateData)` to show the accept/decline UI. No try/catch, no return value.

### showMatchFound
Synchronous, no return value. Reads parameter `debateData`, module state `matchAcceptSeconds`, `screenEl`, and the `MATCH_ACCEPT_SEC` constant. Calls `clearMatchAcceptTimers()` to stop any prior accept countdown. Writes `debateData` into module state via `set_matchFoundDebate`, sets the arena view to `'matchFound'` via `set_view`, then calls `pushArenaState('matchFound')` to push a history entry. Clears `screenEl.innerHTML` if `screenEl` is non-null. Resets the countdown by writing `MATCH_ACCEPT_SEC` via `set_matchAcceptSeconds`. Computes `opInitial` as the uppercase first character of `opponentName` or `'?'`. Creates a `div.arena-match-found.arena-fade-in`, writes an HTML template literal into its `innerHTML` containing the opponent initial, `escapeHTML`-escaped name and topic, raw numeric `opponentElo` (landmine LM-MATCH-002 — no `Number()` cast), the `matchAcceptSeconds` value into `#mf-countdown`, and two buttons `#mf-accept-btn` and `#mf-decline-btn`. Appends the div to `screenEl` via optional chaining. Attaches a click listener to `#mf-accept-btn` that dynamically imports `./arena-match-flow.ts` and invokes its `onMatchAccept()` (fire-and-forget via `void`, promise not awaited). Attaches a click listener to `#mf-decline-btn` that calls `onMatchDecline()`. Reads the current profile via `getCurrentProfile()` and calls `playIntroMusic(profile?.intro_music_id ?? 'gladiator', profile?.custom_intro_url)`. Finally sets a 1000 ms `setInterval` stored via `set_matchAcceptTimer`; each tick decrements the module state by writing `matchAcceptSeconds - 1` via `set_matchAcceptSeconds`, writes the new value's string form into `#mf-countdown` if it exists, and if `matchAcceptSeconds <= 0` calls `onMatchDecline()`. No try/catch.

### onMatchDecline
Synchronous, no parameters, no return value. Calls `clearMatchAcceptTimers()` to stop the accept countdown. Branch: if `isPlaceholder()` is false AND module state `matchFoundDebate` is truthy, fires a `safeRpc('respond_to_match', { p_debate_id: matchFoundDebate.id, p_accept: false })` call; the returned promise is not awaited but has a `.catch` attached that logs `'[Arena] respond_to_match decline failed:'` with the error via `console.warn`. Then calls `returnToQueueAfterDecline()` unconditionally.

### returnToQueueAfterDecline
Synchronous, no parameters, no return value. Writes `null` to module state `matchFoundDebate` via `set_matchFoundDebate(null)`. Branch on `selectedMode`: if truthy, calls `enterQueue(selectedMode, '')` (static import, cycle noted in header comment); otherwise dynamically imports `./arena-lobby.ts` and invokes `renderLobby()` (fire-and-forget via `void`, promise not awaited). No try/catch.

### startAIDebate
Async function, returns `Promise<void>`. Reads parameter `topic` and `AI_TOPICS`, `AI_TOTAL_ROUNDS` constants. Assigns `chosenTopic` to `topic` if truthy, else `randomFrom(AI_TOPICS)`. Seeds local `debateId` with `'ai-local-' + Date.now()`. Branch: if `isPlaceholder()` is false, enters a `try` block that awaits `safeRpc<{ debate_id: string }>('create_ai_debate', { p_category: 'general', p_topic: chosenTopic })`; if the result has no `error` and has `data`, overwrites `debateId` with `data.debate_id`. The `catch` block is empty and falls through to using the local `'ai-local-…'` id (comment: `/* use local */`). After the try/catch (or when `isPlaceholder()` was true and the RPC was skipped), calls `showPreDebate({ id: debateId, topic: chosenTopic, role: 'a', mode: 'ai', round: 1, totalRounds: AI_TOTAL_ROUNDS, opponentName: 'AI Sparring Bot', opponentElo: 1200, ranked: false, ruleset: 'amplified', messages: [] })` fire-and-forget via `void` (its promise is not awaited before the function resolves).

## Agent 02

### clearQueueTimersInline
Synchronous void function with no parameters. Reads module-level `queuePollTimer` and `queueElapsedTimer` from `arena-state.ts`. If `queuePollTimer` is truthy, calls `clearInterval(queuePollTimer)` then writes `null` via `set_queuePollTimer(null)`. If `queueElapsedTimer` is truthy, calls `clearInterval(queueElapsedTimer)` then writes `null` via `set_queueElapsedTimer(null)`. No return value, no branches beyond the two independent `if` checks, no error path.

### onMatchFound
Synchronous void function accepting one parameter `data: MatchData`. Not async. Call order: (1) Invokes `clearQueueTimersInline()` to null out the queue poll/elapsed interval handles. (2) Reads DOM element `arena-queue-status` via `document.getElementById`. If present, writes `textContent` to the target-emoji + "OPPONENT FOUND!" string and writes `style.color` to `var(--mod-accent)`. (3) Schedules a `setTimeout` with 1200 ms delay (fire-and-forget; the outer function returns immediately after scheduling). Inside the timeout callback: reads `data.debate_id`, `data.topic`, `data.role`, `data.total_rounds`, `data.opponent_name`, `data.opponent_id`, `data.opponent_elo`, `data.ruleset`, `data.language` from the parameter. Reads module state `selectedMode`, `selectedRanked`, `selectedRuleset`. Reads constants `DEBATE.defaultRounds` and `AI_TOPICS`. Calls `randomFrom(AI_TOPICS)` when `data.topic` is nullish (via `??`). Builds a local `CurrentDebate` object `debateData` with nullish-coalescing defaults. Branch: if `selectedMode === 'ai'` OR `data.opponent_id` is falsy, calls `enterRoom(debateData)`. Otherwise calls `showMatchFound(debateData)`. No try/catch.

### showMatchFound
Synchronous void function accepting `debateData: CurrentDebate`. Not async. Call order: `clearMatchAcceptTimers()`; `set_matchFoundDebate(debateData)`; `set_view('matchFound')`; `pushArenaState('matchFound')`. If module-level `screenEl` is truthy, writes `screenEl.innerHTML = ''`. `set_matchAcceptSeconds(MATCH_ACCEPT_SEC)`. Computes `opInitial` from `debateData.opponentName[0]` with `'?'` fallback, uppercased. Creates a `div` via `document.createElement('div')`, assigns `className`, then writes `innerHTML` with a template literal interpolating: `opInitial` raw; `escapeHTML(debateData.opponentName)`; `debateData.opponentElo` raw (landmine LM-MATCH-002 — no `Number()` cast); `escapeHTML(debateData.topic)`; `matchAcceptSeconds`. Appends the node with `screenEl?.appendChild(mf)`. Attaches click listener to `#mf-accept-btn` (dynamically imports `./arena-match-flow.ts` → `onMatchAccept()`, fire-and-forget via `void`). Attaches click listener to `#mf-decline-btn` calling `onMatchDecline()`. Reads current profile via `getCurrentProfile()`; calls `playIntroMusic(profile?.intro_music_id ?? 'gladiator', profile?.custom_intro_url)`. Starts `setInterval(..., 1000)` stored through `set_matchAcceptTimer(...)`. Interval tick: decrements via `set_matchAcceptSeconds(matchAcceptSeconds - 1)`; reads `#mf-countdown`; if present writes stringified counter. If `matchAcceptSeconds <= 0`, calls `onMatchDecline()`. Note: interval handle is not cleared inside the tick; clearing falls to `onMatchDecline → clearMatchAcceptTimers`. No try/catch.

### onMatchDecline
Synchronous void, no parameters. Calls `clearMatchAcceptTimers()`. Reads `isPlaceholder()` and module state `matchFoundDebate`. If `!isPlaceholder() && matchFoundDebate` is true, calls `safeRpc('respond_to_match', { p_debate_id: matchFoundDebate.id, p_accept: false })`. Promise not awaited; `.catch((e) => console.warn('[Arena] respond_to_match decline failed:', e))` handler attached. Calls `returnToQueueAfterDecline()`.

### returnToQueueAfterDecline
Synchronous void. Writes `null` via `set_matchFoundDebate(null)`. Branch on `selectedMode`: if truthy, calls `enterQueue(selectedMode, '')` (static cycle per LM-MATCH-001). Else dynamically imports `./arena-lobby.ts` and invokes `renderLobby()` on resolution; promise fire-and-forget via `void`.

### startAIDebate
`async`, returns `Promise<void>`. Reads `topic` param; if falsy, calls `randomFrom(AI_TOPICS)` to produce `chosenTopic`. Initializes `debateId` to `'ai-local-' + Date.now()`. Branch: if `!isPlaceholder()`, enters `try` and awaits `safeRpc<{ debate_id: string }>('create_ai_debate', { p_category: 'general', p_topic: chosenTopic })`. On success, if `!error && data`, overwrites `debateId` with `data.debate_id`. On thrown rejection, `catch { /* use local */ }` swallows silently. If RPC resolves with `error` (non-throw), `debateId` retains local fallback. Calls `showPreDebate(...)` with inline `CurrentDebate` object; return value discarded via `void`.
