## Agent 01

### validatePasswordComplexity

This function is synchronous. It accepts a single `password` string parameter and reads no module-level state, no DOM, and no external state. It runs five sequential conditional checks against `password`, returning a non-null error string at the first check that fails. First it tests `password.length < 8` and returns a minimum-length message if true. Then it applies four regular expression tests in order: `/[a-z]/` for a lowercase letter, `/[A-Z]/` for an uppercase letter, `/[0-9]/` for a digit, and a character-class pattern covering common symbols for a symbol character. Each failing test produces an early return with a specific descriptive string. If all five checks pass, the function returns `null`. There are no loops, no catch blocks, and no calls to any other function or API.

### checkHIBP

This function is async and accepts a single `password` string parameter. It reads no module-level state and touches no DOM. The entire body is wrapped in a `try/catch` block. On the happy path, it first instantiates a `TextEncoder` and calls `encoder.encode(password)` to produce a `Uint8Array`. It then calls `crypto.subtle.digest('SHA-1', data)` and awaits the result, yielding an `ArrayBuffer`. It converts that buffer to a hex string via `Array.from`, `Uint8Array`, `map` with `toString(16).padStart(2,'0')`, and `join('')`, then calls `.toUpperCase()`. It slices the first five characters of that hex string as `prefix` and the remaining characters as `suffix`. Next it creates an `AbortController` and calls `setTimeout` with a 3000 ms delay that invokes `controller.abort()` if fired. It then calls `fetch` with the HIBP range URL `https://api.pwnedpasswords.com/range/${prefix}`, passing `controller.signal` as the abort signal, and awaits the response. Immediately after `fetch` resolves, it calls `clearTimeout` to cancel the abort timer. If `response.ok` is false, it returns `false` without reading the body. Otherwise it awaits `response.text()`, splits the text on `'\n'`, and calls `.some()` on the resulting lines, checking whether any line's content before the first `:` (trimmed) equals `suffix`. The return value of `.some()` — a boolean indicating whether the password hash suffix appears in the breach list — is the function's return value on the happy path. The `catch` block accepts no binding and returns `false` unconditionally, covering both network failures and the `AbortError` thrown when the 3-second timeout fires.
