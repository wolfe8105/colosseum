### trackEvent

The entire body is wrapped in a `try/catch` that silently discards any exception. At the top of the try block it calls `isOptedOut()` and returns immediately if that returns true. Otherwise it calls `getVisitorId()`, `getTrafficSource()`, and `getUserId()` — all imported from `analytics.utils` — to read module-level or localStorage state external to this file. If `extra` is a non-null object, it iterates `Object.keys(extra)` and copies each key-value pair onto `metadata`, allowing callers to attach arbitrary fields. It then builds a `body` object with `p_event_type` and `p_metadata`; if `getUserId()` returned a truthy value, `p_user_id` is also appended. It calls the native `fetch()` API — not `supabase.rpc()` — as a fire-and-forget POST to `SUPABASE_URL + '/rest/v1/rpc/log_event'`, passing `SUPABASE_ANON_KEY` in both the `apikey` and `Authorization` headers. The `fetch()` call is not awaited; its returned promise has `.catch()` chained to it, which logs a warning to `console.warn` if the network request rejects. Any synchronous exception thrown before or after the `fetch()` call is caught by the outer `try/catch` and discarded without logging. The function returns `undefined` in all paths.

### checkSignup

`checkSignup` is a synchronous, void function that takes no parameters. Its entire body is wrapped in a `try/catch` that silently discards any exception. It first calls `isOptedOut()` and returns early if true. It then calls `migrateKeys()` (an imported utility whose side effects are not visible in this file) before calling `getUserId()` to read the current authenticated user ID. It reads `mod_uid_seen` from `localStorage`. The function then branches on two conditions. If `userId` is truthy and `wasSeen` is falsy (the key was absent or null), it writes `userId` to `localStorage` under `mod_uid_seen` and calls `trackEvent('signup', { new_user: true })`. The outer `catch` block contains only a comment and produces no output, so any exception from `migrateKeys`, `getUserId`, `localStorage`, or `trackEvent` is silently swallowed. The function returns `undefined` in all paths.
