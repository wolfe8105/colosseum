# Stage 2 Outputs — webrtc.ice.ts

## Agent 01

### fetchTurnCredentials
This async function fetches short-lived TURN credentials from a Supabase Edge Function. It reads the access token via `getAccessToken()` and makes a POST request to `${SUPABASE_URL}/functions/v1/turn-credentials` with the token in an Authorization header. If no token is available, it returns null immediately. If the HTTP response status is not ok, it logs a warning and returns null. The function then parses the response body as JSON and validates that it contains an `iceServers` property that is an array; if validation fails, it logs a warning and returns null. On successful validation, it logs a success message and returns the `iceServers` array cast as `RTCIceServer[]`. If any error occurs during the fetch or JSON parsing, the catch block logs a warning with the error and returns null.

### getIceServers
This async exported function returns ICE servers for WebRTC connections, using either cached TURN credentials or a fallback. It first checks the module-level `state.fetchedIceServers`; if already set, it returns that value immediately without async work. If not cached, the function checks `state.turnFetchPromise` to detect whether a fetch is already in flight; if not, it initiates a new fetch by calling `fetchTurnCredentials()` and stores the returned promise in `state.turnFetchPromise`. The function then awaits the promise stored in `state.turnFetchPromise`, clears that state variable to null, and evaluates the result. If the result is truthy, it stores it in `state.fetchedIceServers`, and returns it. If the result is falsy (indicating `fetchTurnCredentials` returned null), the function returns the module-level `FALLBACK_ICE_SERVERS` instead.

On successful validation, it logs to the console, casts `body.iceServers` to `RTCIceServer[]`, and returns that array. The function then awaits the promise stored in `state.turnFetchPromise` and assigns the result to a local variable. It immediately writes null to `state.turnFetchPromise` to clear the deduplication flag. If the awaited result is truthy, it writes the result to `state.fetchedIceServers` and returns it. If the result is falsy, the function returns `FALLBACK_ICE_SERVERS`, which is imported module-level state.
